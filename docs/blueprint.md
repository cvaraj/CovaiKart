# **App Name**: KonguMart

## Core Features:

- Seller Registration with Geolocation: Allow sellers to register, validating their business address is within the Coimbatore region using PIN code validation and geolocation. Admin KYC approval workflow is included.
- Product Listing and Search: Enable sellers to list products with variants (size, color), images, descriptions, and inventory. Buyers can search for products with autocomplete, typo tolerance, and filters.
- AI-Powered Product Recommendation Tool: Uses an AI tool to recommend products based on user browsing history, search queries, and purchase patterns.
- Secure Multi-Vendor Checkout: Implements a secure checkout process with multiple payment options (Razorpay, PayU, COD). Buyers can manage addresses, payment methods, and apply coupons.
- Order Management and Fulfillment: Sellers can view, process, and fulfill orders. Buyers can track shipments, view order history, and initiate returns.
- Admin Moderation: Admin dashboard with role-based access control. Enables approving/rejecting sellers and products, managing commissions and resolving disputes.
- Seller Analytics and Reporting: Provide sellers with analytics on sales, inventory, and customer behavior to improve their marketplace performance.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to represent trust and reliability.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Orange (#F2994A) for calls to action and highlights to draw attention to important elements.
- Headline font: 'Poppins' (sans-serif) for headings. It offers a clean and modern appearance.
- Body font: 'PT Sans' (sans-serif) for readability, particularly in longer text blocks.
- Use a consistent set of line icons to visually represent product features and navigation options. All icons should maintain a cohesive, minimalist design.
- Implement a responsive grid system to adapt the layout across devices.
